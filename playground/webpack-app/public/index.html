Webpack App with Faker Plugin

Test API Request

Click the button below to fetch data from /api/users.

Make sure you have configured a mock for this endpoint in the Faker UI.

Instructions

  1. Look for the Faker UI button in the bottom right corner (or where configured).
  2. Click it to open the dashboard.
  3. Create a new Mock:
    • URL: /api/users
    • Method: GET
    • Response Body: [{"id": 1, "name": "John Doe"}]
  4. Enable the mock and try the Fetch button above.
